<task type="auto">
  <name>Task 2: Add M3 shape scale and elevation utilities</name>
  <files>frontend/tailwind.config.ts</files>
  <read_first>frontend/tailwind.config.ts</read_first>
  <action>
**Replace the existing `borderRadius` extension** with M3 shape scale tokens. Replace:
```typescript
borderRadius: {
  lg: 'var(--radius)',
  md: 'calc(var(--radius) - 2px)',
  sm: 'calc(var(--radius) - 4px)'
}
```

With:
```typescript
borderRadius: {
  none: '0px',
  xs: '4px',
  sm: '8px',
  md: '12px',
  lg: '16px',
  xl: '28px',
  full: '9999px',
  DEFAULT: 'var(--radius)',
},
```

The `DEFAULT` value references `--radius` (0.75rem = 12px) for backward compatibility with components using `rounded` without a size suffix.

**Add elevation colors** to the `colors` extension (append to existing colors object):
```typescript
elevation: {
  '1': 'hsl(var(--elevation-1))',
  '2': 'hsl(var(--elevation-2))',
  '3': 'hsl(var(--elevation-3))',
},
```

**Add new semantic colors** for M3-specific roles to the colors extension:
```typescript
'primary-container': {
  DEFAULT: 'hsl(var(--primary-container))',
  foreground: 'hsl(var(--on-primary-container))',
},
'secondary-container': {
  DEFAULT: 'hsl(var(--secondary-container))',
  foreground: 'hsl(var(--on-secondary-container))',
},
'surface-tint': 'hsl(var(--surface-tint))',
outline: {
  DEFAULT: 'hsl(var(--outline))',
  variant: 'hsl(var(--outline-variant))',
},
```
  </action>
  <verify>
    <automated>grep "xs: '4px'" frontend/tailwind.config.ts && grep "xl: '28px'" frontend/tailwind.config.ts && grep "elevation" frontend/tailwind.config.ts && grep "primary-container" frontend/tailwind.config.ts</automated>
  </verify>
  <acceptance_criteria>

    - tailwind.config.ts borderRadius has `DEFAULT: 'var(--radius)'`
    - tailwind.config.ts colors has `elevation` with keys '1', '2', '3'
    - tailwind.config.ts colors has `'primary-container'` with DEFAULT and foreground
    - tailwind.config.ts colors has `'secondary-container'` with DEFAULT and foreground

  </acceptance_criteria>
