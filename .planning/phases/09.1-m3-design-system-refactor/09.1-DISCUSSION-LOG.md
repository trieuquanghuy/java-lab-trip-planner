## Color Mapping Strategy

| Option | Description | Selected |
|--------|-------------|----------|
| Keep shadcn names, swap values + add new M3 vars | Minimal component changes, extends system | ✓ |
| Rename everything to M3 roles | Full M3 purity, breaks all component references | |
| Only change values, no new vars | Simplest, but loses M3 container/surface roles | |

**User's choice:** Agent decision — keep shadcn names, change values to M3 palette, add new variables for M3-specific roles (primary-container, surface-tint, outline, etc.)
**Notes:** Maps `--muted` → surface-variant, `--accent` → secondary-container, `--card` → surface. Maintains zero-change contract for existing shadcn components.

## Dark Mode Toggle

| Option | Description | Selected |
|--------|-------------|----------|
| System preference auto-detect + manual override in nav | Best UX, respects OS preference, allows user control | ✓ |
| System preference only (no toggle UI) | Simpler, but no user control | |
| Manual toggle only (ignore system) | User always must choose | |

**User's choice:** Agent decision — auto-detect + manual override. Sun/moon Lucide icon in Layout header. localStorage persistence with "system" default.
**Notes:** Inline `<script>` in index.html prevents flash of wrong theme on load.

## Palette Generation Method

| Option | Description | Selected |
|--------|-------------|----------|
| Hardcode HSL from M3 theme builder output | No runtime dep, static values, lean bundle | ✓ |
| Use @material/material-color-utilities at build time | Accurate, regenerable, adds dep | |
| Use @material/material-color-utilities at runtime | Dynamic theming possible, heaviest | |

**User's choice:** Agent decision — hardcode HSL values from M3 theme builder. Seed is static, no need for runtime generation. Regenerate manually if seed ever changes.
**Notes:** Portfolio project — runtime theming is overkill.

## Typography Scope

| Option | Description | Selected |
|--------|-------------|----------|
| Map only 7 used variants | Pragmatic, no unused utilities | ✓ |
| Map all 14 M3 variants | Complete M3 coverage, many unused | |
| Map 4 minimal variants (heading, body, label, small) | Too coarse for M3 compliance | |

**User's choice:** Agent decision — 7 variants: headline-lg, title-lg, title-md, body-lg, body-md, label-lg, label-sm. Covers page titles, section headers, card titles, descriptions, regular text, buttons, and metadata.
**Notes:** Font family stays system sans-serif. Defined as Tailwind theme extension utilities.
