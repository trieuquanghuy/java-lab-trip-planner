**Replace animation durations and easings:**
```css
/* Fade-in animation */
.animate-fade-in {
  animation: fade-in var(--duration-medium) var(--easing-standard) both;
}

/* Slide-up animation */
.animate-slide-up {
  animation: slide-up var(--duration-medium) var(--easing-emphasized-decelerate) both;
}

/* Scale-in animation */
.animate-scale-in {
  animation: scale-in var(--duration-short) var(--easing-standard) both;
}

/* Stagger children */
.stagger-children > * {
  animation: slide-up var(--duration-medium) var(--easing-emphasized-decelerate) both;
}
```

Keep stagger delay values unchanged (0ms, 50ms, 100ms, etc.).

**Update shimmer animation** — keep the 1.5s infinite timing (it's a background loop, not an interaction animation).

**Add prefers-reduced-motion media query** at the end of the file (before closing `@layer base`):
```css
@media (prefers-reduced-motion: reduce) {
  .animate-fade-in,
  .animate-slide-up,
  .animate-scale-in,
  .stagger-children > * {
    animation: none !important;
    opacity: 1 !important;
    transform: none !important;
  }
  .animate-shimmer {
    animation: none !important;
  }
}
```
