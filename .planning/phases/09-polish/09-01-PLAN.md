<task type="auto">
  <name>Task 2: Code-split routes with React.lazy + wrap with ErrorBoundary</name>
  <files>frontend/src/App.tsx</files>
  <read_first>frontend/src/App.tsx, frontend/src/components/ErrorBoundaryFallback.tsx, frontend/src/pages/PageLoader.tsx</read_first>
  <action>
Refactor `frontend/src/App.tsx`:

1. Remove static imports of all page components EXCEPT HomePage (eager load).
2. Add React.lazy imports for each page:
   ```
   const LoginPage = lazy(() => import('@/pages/LoginPage').then(m => ({ default: m.LoginPage })));
   const SignupPage = lazy(() => import('@/pages/SignupPage').then(m => ({ default: m.SignupPage })));
   const VerifyEmailPage = lazy(() => import('@/pages/VerifyEmailPage').then(m => ({ default: m.VerifyEmailPage })));
   const DestinationDetailPage = lazy(() => import('@/pages/DestinationDetailPage').then(m => ({ default: m.DestinationDetailPage })));
   const TripsPage = lazy(() => import('@/pages/TripsPage').then(m => ({ default: m.TripsPage })));
   const TripDetailPage = lazy(() => import('@/pages/TripDetailPage').then(m => ({ default: m.TripDetailPage })));
   const NotFoundPage = lazy(() => import('@/pages/NotFoundPage').then(m => ({ default: m.NotFoundPage })));
   const ServerErrorPage = lazy(() => import('@/pages/ServerErrorPage').then(m => ({ default: m.ServerErrorPage })));
   ```
3. Import `{ lazy, Suspense }` from react, `{ ErrorBoundary }` from react-error-boundary
4. Import `{ ErrorBoundaryFallback }` and `{ PageLoader }`
5. Wrap the entire `<Routes>` block with:
   ```
   <ErrorBoundary FallbackComponent={ErrorBoundaryFallback}>
     <Suspense fallback={<PageLoader />}>
       <Routes>...</Routes>
     </Suspense>
   </ErrorBoundary>
   ```
6. Add `/error` route: `<Route path="/error" element={<ServerErrorPage />} />`
7. Keep `HomePage` as a direct import (eager loaded per UI-SPEC).
8. Remove the PlaceholderPage function if it exists.
  </action>
