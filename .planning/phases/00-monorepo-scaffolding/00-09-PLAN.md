Output: A `frontend/` pnpm project that builds (`pnpm build`), tests pass (`pnpm test --run` runs the App.test.tsx smoke), lints clean (`pnpm lint`), renders `http://localhost:5173` with zero browser console errors, AND a multi-stage Dockerfile that lets `docker compose up --wait` bring the frontend container to ready without any prior `pnpm build` step.
