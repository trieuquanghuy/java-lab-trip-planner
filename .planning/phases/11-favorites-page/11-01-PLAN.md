3. Add a `findAllById`-based lookup to the repository. The `DestinationsCacheRepository` already extends `JpaRepository<DestinationsCacheEntity, String>` so `findAllById(List<String>)` is inherited. No repository changes needed ‚Äî just use the inherited method.

4. Create `BatchService.java`:
   ```java
   @Service
   public class BatchService {
       private final DestinationsCacheRepository cacheRepository;
       private final ProviderMapper mapper;

       // Constructor injection

       public BatchResponse getBatch(List<String> refs) {
           List<DestinationsCacheEntity> entities = cacheRepository.findAllById(refs);
           List<NearbyItem> items = entities.stream()
               .map(mapper::toNearbyItem)
               .toList();
           return new BatchResponse(items);
       }
   }
   ```
   Check if `ProviderMapper` already has a `toNearbyItem` method. If not, add one that maps entity fields to `NearbyItem(entity.getProviderRef(), entity.getName(), entity.getCategory(), entity.getRating(), entity.getLat(), entity.getLng(), /* photoUrl from photos JSON ‚Äî first photo or null */)`. For photoUrl: parse the `photos` field (stored as JSON array string) and extract the first URL, or return null if empty/null.

5. Create `BatchController.java`:
   ```java
   @RestController
   @RequestMapping("/api/destinations")
   public class BatchController {
       private final BatchService batchService;

       // Constructor injection

       @PostMapping("/batch")
       public ResponseEntity<BatchResponse> postBatch(@RequestBody BatchRequest request) {
           if (request == null || request.refs() == null || request.refs().isEmpty()) {
               return ResponseEntity.ok(new BatchResponse(List.of()));
           }
           if (request.refs().size() > 50) {
               return ResponseEntity.badRequest().build();
           }
           // Validate each ref matches pattern
           for (String ref : request.refs()) {
               if (ref == null || !ref.matches("^(otm|fsq):[a-zA-Z0-9_-]+$")) {
                   return ResponseEntity.badRequest().build();
               }
           }
           BatchResponse response = batchService.getBatch(request.refs());
           return ResponseEntity.ok(response);
       }
   }
   ```
   Key: max 50 refs (DoS protection per threat model), same ref validation regex as DetailController, empty list returns empty response (not error).
