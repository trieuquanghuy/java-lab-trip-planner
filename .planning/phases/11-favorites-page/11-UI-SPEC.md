## Interaction Specification

### Unfavorite flow
1. User clicks filled heart button (top-right corner of card)
2. Card immediately starts fade-out (`opacity-0 transition-opacity duration-200`)
3. After 200ms transition completes, card is removed from DOM (via React state)
4. `DELETE /api/favorites/{ref}` fires in background
5. On error: card reappears (revert optimistic update), error toast shown

### Card click (navigation)
- Clicking anywhere on the card except the unfavorite button navigates to `/destinations/{ref}`
- Uses existing `<Link>` behavior from DestinationCard

### Loading state
- Show 6 skeleton cards in the same grid layout
- Skeletons match DestinationCard dimensions (photo area + content area)

### Empty state
- Centered vertically in available space
- Heart icon in primary/10 circle (80x80px container, 40x40 icon)
- CTA links to home page (`/`)

## Accessibility

| Requirement | Implementation |
|------------|----------------|
| WCAG AA contrast | All text meets 4.5:1 against backgrounds (inherited from M3 system) |
| Touch target | Unfavorite button minimum 44x44px (`h-9 w-9` = 36px — add `touch-target` utility class or increase to `h-11 w-11`) |
| Focus visible | Unfavorite button shows focus ring on keyboard navigation (`focus-visible:ring-2 ring-ring`) |
| Screen reader | Unfavorite button has `aria-label="Remove from favorites"` |
| Reduced motion | Fade-out animation respects `prefers-reduced-motion` (existing CSS rule in index.css) |
| Live region | When unfavorite succeeds, announce "Removed from favorites" via `aria-live="polite"` (or toast with role="status") |
| Heading hierarchy | `<h1>` for "My Favorites", `<h2>` for empty state "No favorites yet" |

## Animation & Motion

| Element | Trigger | Duration | Easing | Property |
|---------|---------|----------|--------|----------|
| Page enter | Route mount | 300ms | `--easing-standard` | opacity (fade-in) |
| Card removal | Unfavorite click | 200ms | `--easing-standard` | opacity 1→0 |
| Card appear (rollback) | API error | 200ms | `--easing-emphasized-decelerate` | opacity 0→1 |
| Loading shimmer | While fetching | 1500ms loop | linear | background-position |

All animations gated by `prefers-reduced-motion: reduce` media query (already handled in index.css).
